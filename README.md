# Plancha360 Website

A premium-accessible outdoor cooking brand site focused on social cooking, waitlist conversion, and later direct sales.

## Current decision
- **Framework:** Astro
- **Primary CTA:** Join waitlist
- **Brand direction:** premium / minimal with warm Mediterranean / social energy
- **Launch scope:** one hero product, no supplier-facing content

## Why Astro
- strong SEO
- clean content management with Markdown/MDX
- flexible custom design
- easy to expand later for blog and ecommerce

## Next implementation steps
1. scaffold Astro project
2. define design tokens and layout system
3. build homepage
4. add product, FAQ, and waitlist sections
5. wire up content collection for future SEO pages
6. add analytics + form handling

## Content model
- homepage copy
- product facts
- FAQ
- waitlist form
- later: SEO articles and ecommerce pages

## Notes
- Visuals are currently limited
- Best approach: strong typography + layout first, imagery second
- Use the repo for source-controlled website work
