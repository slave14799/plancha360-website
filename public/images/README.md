# Plancha360 image folder structure

Put the final website images in these folders and use the naming pattern below.

## Folders

- `logo/` — brand logo files
- `hero/` — main homepage hero images
- `products/` — product detail shots and model images
- `lifestyle/` — social / cooking / atmosphere images
- `rental/` — future event / rental images

## Recommended filenames

### Logo
- `plancha360-logo.png`
- `plancha360-logo.svg`

### Hero
- `hero-main.jpg`
- `hero-secondary.jpg`

### Products
- `model-900-01.jpg`
- `model-900-02.jpg`
- `model-700-01.jpg`
- `model-700-02.jpg`
- `detail-fire.jpg`
- `detail-material.jpg`

### Lifestyle
- `lifestyle-01.jpg`
- `lifestyle-02.jpg`
- `lifestyle-03.jpg`

### Rental / events later
- `event-01.jpg`
- `event-02.jpg`

## Simple rule

If you are unsure where an image belongs:
- **hero** = best first impression image
- **products** = product-focused shots
- **lifestyle** = people, gatherings, food, atmosphere
- **logo** = brand mark files

These files are served directly from `/images/...` in the site.
